#📱 React Phone Catalog

A responsive phone catalog web application built with React.
The project simulates a real e-commerce interface with product listing, filtering, product details pages, and a clean modern UI. It focuses on scalable component architecture and best practices in React development.

## 🚀 Features
Responsive layout (desktop, tablet, mobile)
Product catalog with reusable components
Product details page with dynamic routing (React Router)
Sorting and filtering functionality
Clean and scalable component structure
SCSS Modules for modular styling
State management using React hooks
Optimized UI structure for performance and reusability
🛠️ Tech Stack
React
JavaScript (ES6+)
React Router
SCSS (Sass)
HTML5
npm / yarn
🎯 Purpose

This project was created to practice building real-world React applications, improve component architecture, and simulate an online store with dynamic routing and state-driven UI.
